# Security Policies

## Contexto
Biblioteca oficial de políticas de segurança da informação, privacidade e compliance para todas as unidades Ávila.

## Objetivo
Manter diretrizes atualizadas para hardening, classificação de dados, resposta a incidentes e requisitos regulatórios.

## Estrutura Recomendada
- `policies/` — documentos formais (PDF/Markdown) aprovados pela governança.
- `standards/` — guias complementares e checklists operacionais.
- `audits/` — relatórios de auditoria interna/externa.
- `training/` — materiais de conscientização e certificações obrigatórias.

## Rotinas Essenciais
- Revisar políticas semestralmente com Lex (compliance) e Atlas.
- Sincronizar mudanças com tarefas do Archivus para garantir conformidade.
- Registrar aceitações e comunicados aos times em `Docs/Relatorios/Avisos`.
