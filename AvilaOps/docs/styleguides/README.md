# Style Guides

## Contexto
Coleção de guias de estilo (código, documentação, UI, dados) que padronizam entregas entre squads Ávila.

## Objetivo
Disponibilizar padrões oficiais que devem ser aplicados em novos projetos, PRs e materiais corporativos.

## Estrutura Recomendada
- `code/` — convenções para linguagens (Python PEP8 Ávila, TypeScript, PowerShell).
- `docs/` — templates Markdown, glossário, terminologia.
- `design/` — guias visuais, cores, tipografia.
- `data/` — padrões de nomenclatura, schemas e métricas.

## Rotinas Essenciais
- Atualizar snippets VS Code (`avila-*`) conforme guias evoluem.
- Referenciar estes padrões em `.github/copilot-instructions.md`.
- Revisar trimestralmente com Atlas/Echo (branding) e Helix (engenharia).

## Responsável
- Atlas Squad — Estratégia/Corporativo (com apoio Echo)
