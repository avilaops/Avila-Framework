# Insight

## Contexto
Produto **Insight**, plataforma de analytics avançado para diretoria e squads corporativos, integrada ao ecossistema `data/`.

## Objetivo
Centralizar materiais técnicos, modelos analíticos, integrações e UI associados ao Insight.

## Estrutura Recomendada
- `docs/` — visão funcional, personas, jornadas.
- `analytics/` — notebooks, modelos de previsão, relatórios.
- `frontend/` — assets de interface e componentes.
- `backend/` — APIs e serviços de dados.
- `roadmap/` — metas trimestrais e KPIs.
