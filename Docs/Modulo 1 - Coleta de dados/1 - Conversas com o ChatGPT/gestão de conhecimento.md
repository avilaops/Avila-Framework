AvilaInc/governance_framework/ como vault.

Tudo que for documentação estratégica, aprendizado, decisões, normas e revisões vive ali.

Templates e JSON do Web Clipper já estão prontos para capturar dados.

Use Dataview e Templater para conectar documentos a métricas e setores.

Automatize backups do vault via Git:

cd AvilaInc/governance_framework
git init
git add .
git commit -m "Governance vault initialization"

Assim você versiona também as “leis” e revisões internas.
